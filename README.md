# PAE-Clojure
Esse projeto se trata de uma API REST feita em Clojure, usando o paradigma da programação funcional. O projeto se trata da criação de uma página web que calcula os trastes para interessados na compra de um novo violão em uma loja específica. Esse é um projeto extensionista com o foco de produzir um produto para uma empresa de fora da faculdade.
